# Pascoal

> Un IDE moderno de Pascal para escritorio: escribe, compila y ejecuta programas en Pascal.

[English](../../README.md) · [Português (BR)](pt-BR.md) · [Polski](pl.md)

---

## ¿Qué es Pascoal?

Pascoal es un IDE de Pascal para escritorio desarrollado con Tauri y Rust, diseñado para acercar la programación en Pascal a una audiencia moderna. Interfaz limpia, compilación real mediante Free Pascal (FPC), consola interactiva para programas, explorador de archivos, búsqueda entre archivos e integración con Git: todo en un paquete liviano.

Nació como una rebelión contra herramientas anticuadas como Pascalzim, con el objetivo de hacer que Pascal vuelva a ser accesible y divertido, sin el peso de Electron ni el desorden de interfaces heredadas.

## Funcionalidades

* **Editor moderno** impulsado por CodeMirror 6, con resaltado de sintaxis para Pascal y temas reactivos
* **Compilación real** mediante Free Pascal Compiler (FPC)
* **Consola interactiva** con zonas separadas para la compilación y la salida del programa: `readln` funciona
* **Edición con múltiples pestañas**: abre varios archivos al mismo tiempo
* **Explorador de archivos**: abre una carpeta y navega por sus archivos Pascal
* **Buscar y reemplazar**: widget flotante al estilo VSCode con resaltado de coincidencias, además de búsqueda entre archivos dentro de una carpeta abierta
* **Integración con Git**: consulta el estado, haz stage/unstage, inspecciona diffs, crea commits e inicializa repositorios, todo desde un panel dedicado
* **Menú nativo del sistema**: menús Archivo y Ayuda integrados en la barra de título, con enlaces directos para reportar errores o solicitar funciones en GitHub
* **Tres temas**: Oscuro, Claro y Charcoal, con detección del sistema
* **Controles nativos de ventana** adaptados por plataforma (traffic lights en macOS, estilo Windows/Linux)
* **Guardado automático antes de ejecutar**: configurable
* **Interfaz internacionalizada**: English, Português (BR), Español (Latinoamérica) y Polski, con persistencia de la selección de idioma
* **Liviano**: ~25 MB de RAM, instaladores de ~5 MB

## Requisitos

* [Rust](https://rustup.rs/) (stable)
* [Node.js](https://nodejs.org/) >= 22
* [Free Pascal Compiler (FPC)](https://www.freepascal.org/download.html) instalado y disponible en el PATH
* [Git](https://git-scm.com/) instalado y disponible en el PATH (necesario para el panel de Git)
* [Requisitos previos de Tauri](https://tauri.app/start/prerequisites/) para tu plataforma

## Primeros pasos

```bash
# Clonar el repositorio
git clone https://github.com/brener-fregulia/Pascoal.git
cd Pascoal

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
cargo tauri dev

# Generar build de producción
cargo tauri build
```

### Solo frontend (desarrollo de UI sin Tauri)

```bash
npm run dev:ide
```

### Consejos de desarrollo

Para acelerar los rebuilds de Rust durante el desarrollo, instala [sccache](https://github.com/mozilla/sccache):

```bash
cargo install sccache
```

Luego configúralo como wrapper del compilador de Rust en `src-tauri/.cargo/config.toml`:

```toml
[build]
rustc-wrapper = "sccache"
```

Esto también se usa en CI para acelerar los builds en los runners de GitHub Actions.

### Ejecutar las pruebas

```bash
npm test              # frontend + Rust + Pascal
npm run test:frontend # solo Vitest
npm run test:rust     # solo cargo test
npm run test:pascal   # pruebas de integración de Pascal (requiere FPC)
```

## Estructura del proyecto

```text
src/
  renderer/
    ide/                    # Frontend en Svelte + Vite
      src/
        components/         # Componentes Svelte (Titlebar, TabBar, Editor, Console, FileTree, SearchPanel, GitPanel, FindWidget, AboutModal...)
        icons/              # Componentes de íconos SVG
        stores/             # Stores de Svelte (tabs, theme, console, runner, settings, explorerStore, searchStore, gitStore...)
        i18n/               # Archivos de locale y store de traducciones
        styles/             # CSS global
src-tauri/
  src/
    lib.rs                  # Configuración de la app y registro de comandos
    env.rs                  # Detección de FPC y directorio de documentos
    fs.rs                   # Comandos de E/S de archivos, explorador de carpetas y búsqueda entre archivos
    git.rs                  # Comandos de estado, stage, diff, commit e init de Git
    compiler.rs             # Lógica de compilación con FPC
    process.rs              # Estado de procesos, run_with_pipes, run_with_pty
    tests/                  # Pruebas unitarias de Rust
  tauri.conf.json
  Cargo.toml
tests/
  frontend/                 # Pruebas con Vitest
  pascal/                   # Pruebas de integración en Pascal y scripts
docs/
  readme/                   # Traducciones del README
scripts/
  set-version.cjs           # Script para actualizar la versión
```

## Stack tecnológico

|                      |                              |
| -------------------- | ---------------------------- |
| Runtime              | Tauri 2                      |
| Backend              | Rust                         |
| Frontend             | Svelte 5 + Vite + TypeScript |
| Compilador           | Free Pascal (FPC)            |
| Editor               | CodeMirror 6                 |
| Consola              | xterm.js                     |
| Control de versiones | Git (vía CLI)                |
| Pruebas              | Vitest + cargo test          |
