# Pascoal

> Un IDE Pascal moderno para escritorio: escribe, compila y ejecuta programas Pascal.

[English](../../README.md) · [Português (BR)](pt-BR.md)

---

## ¿Qué es Pascoal?

Pascoal es un IDE Pascal de escritorio construido con Tauri y Rust, diseñado para acercar la programación en Pascal a una audiencia moderna. Interfaz limpia, compilación real mediante Free Pascal (FPC) y una consola interactiva con salida separada para compilación y programa: todo en un paquete liviano.

Nació como una rebelión contra herramientas desactualizadas como Pascalzim, con el objetivo de hacer que Pascal vuelva a ser accesible y divertido, sin el peso de Electron ni el desorden de las interfaces heredadas.

## Funcionalidades

* **Editor moderno** con CodeMirror 6, resaltado de sintaxis Pascal y temas reactivos
* **Compilación real** mediante Free Pascal Compiler (FPC)
* **Consola interactiva** con zonas separadas para salida de compilación y salida del programa — `readln` funciona
* **Edición en múltiples pestañas** — abre varios archivos simultáneamente
* **Tres temas** — Dark, Light y Charcoal, con detección automática del sistema
* **Controles de ventana nativos** adaptados por plataforma (traffic lights en macOS, estilo Windows/Linux)
* **Guardado automático antes de ejecutar** — configurable
* **Liviano** — ~25 MB de RAM, instaladores de ~5 MB

## Requisitos

* [Rust](https://rustup.rs/) (stable)
* [Node.js](https://nodejs.org/) >= 22
* [Free Pascal Compiler (FPC)](https://www.freepascal.org/download.html) instalado y disponible en el PATH
* [Prerequisitos de Tauri](https://tauri.app/start/prerequisites/) para tu plataforma

## Cómo empezar

```bash
# Clona el repositorio
git clone https://github.com/brener-fregulia/Pascoal.git
cd Pascoal

# Instala las dependencias
npm install

# Ejecuta en modo de desarrollo
cargo tauri dev

# Compila para producción
cargo tauri build
```

### Solo frontend

Desarrollo de UI sin Tauri:

```bash
npm run dev:ide
```

### Ejecutar pruebas

```bash
npm test              # frontend + Rust + Pascal
npm run test:frontend # solo Vitest
npm run test:rust     # solo cargo test
npm run test:pascal   # pruebas de integración Pascal (requiere FPC)
```

## Estructura del proyecto

```txt
src/
  renderer/
    ide/                 # Frontend Svelte + Vite
      src/
        components/      # Componentes Svelte (Titlebar, TabBar, Editor, Console...)
        icons/           # Componentes de íconos SVG
        stores/          # Stores Svelte (tabs, theme, console, runner, settings)
        styles/          # CSS global

src-tauri/
  src/
    lib.rs               # Configuración de la app y registro de comandos
    env.rs               # Detección de FPC y directorio de documentos
    fs.rs                # Comandos de I/O de archivos
    compiler.rs          # Lógica de compilación con FPC
    process.rs           # Estado del proceso, run_with_pipes, run_with_pty
  tests/                 # Pruebas unitarias Rust
  tauri.conf.json
  Cargo.toml

tests/
  frontend/              # Pruebas Vitest
  pascal/                # Pruebas de integración Pascal y scripts

docs/
  readme/                # Traducciones del README

scripts/
  set-version.cjs        # Script de bump de versión
```

## Stack tecnológica

|            |                              |
| ---------- | ---------------------------- |
| Runtime    | Tauri 2                      |
| Backend    | Rust                         |
| Frontend   | Svelte 5 + Vite + TypeScript |
| Compilador | Free Pascal (FPC)            |
| Editor     | CodeMirror 6                 |
| Consola    | xterm.js                     |
| Pruebas    | Vitest + cargo test          |
